🚀 DEPLOY VERCEL — PASSO A PASSO
1
Vai a github.com → login com Google → cria repo botfx (Public)
2
No repo vazio toca "uploading an existing file" → faz upload dos 3 ficheiros: vercel.json, bitget.js, index.html
3
Vai a vercel.com → login Google → Add New Project → liga ao repo botfx → Deploy
4
Vercel → Settings → Environment Variables → adiciona:
BITGET_API_KEY
BITGET_API_SECRET
BITGET_PASSPHRASE
5
Toca Redeploy → copia o URL → cola acima
⚠️ A Bitget exige uma Passphrase além da API Key e Secret — é uma palavra-passe que defines tu quando crias a API Key na Bitget.
🔑 CRIAR API KEY NA BITGET
1
Bitget → Perfil → API Management → Create API
2
Nome: botfx → define uma Passphrase (guarda nas Notas)
3
Ativa ✅ Futures Trading — mais nada
4
NÃO ativas Withdrawals
5
Copia API Key, Secret Key e a tua Passphrase para as Notas
